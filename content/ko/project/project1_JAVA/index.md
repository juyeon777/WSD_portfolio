- **진행 기간:** 2023-03-01 ~ 2023-06-16
- **기술 스택:** Java, Java Swing
- 자바를 이용해 넷플릭스 '오징어 게임'에 나오는 줄다리기 게임을 재현했습니다.
- 2023학년도 1학기 '객체지향프로그래밍' 과목에서 진행한 팀 프로젝트입니다.

**담당 역할 — 본인 담당**

- 기획: 팀원들과 함께 게임 컨셉·규칙 논의에 참여
- **UI·그래픽 디자인 (메인 기여):** 캐릭터를 비롯한 게임 내 모든 그래픽 요소를 **픽셀 아트로 직접 디자인·제작**하여 게임의 비주얼 전반을 담당
- 프론트엔드 구현: 버튼 등 UI 요소 구현
