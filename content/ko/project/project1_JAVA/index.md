## 프로젝트 상세 — 문제 → 해결 → 결과

- **게임 비주얼 (UI·그래픽):** 줄다리기 게임에 어울리는 그래픽 자산이 필요 → 캐릭터를 비롯한 **모든 그래픽 요소를 픽셀 아트로 직접 제작** → 게임의 비주얼 정체성을 완성하고 몰입감을 높임
- **화면 상호작용 (프론트엔드):** 사용자가 게임을 조작할 수단 필요 → **버튼 등 UI 요소를 구현** → 플레이어가 직접 게임을 진행할 수 있게 함
- **컨셉 설계 (기획):** 팀이 같은 그림을 공유해야 함 → 팀원들과 게임 컨셉·규칙 논의에 참여 → 디자인 방향과 게임 규칙을 함께 확정

## 담당 역할 — 본인 담당

- 기획: 팀원들과 게임 컨셉·규칙 논의 참여
- **UI·그래픽 디자인 (메인 기여):** 캐릭터를 비롯한 게임 내 모든 그래픽 요소를 픽셀 아트로 직접 디자인·제작
- 프론트엔드 구현: 버튼 등 UI 요소 구현

## 주 업무 외 기여

- 디자인·구현 외에도 게임 컨셉·규칙 논의에 참여해 팀의 방향을 맞추는 데 기여
