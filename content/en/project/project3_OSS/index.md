## Details — Problem → Solution → Result

- **Empty template (impl):** The chosen template had no menu tabs or base code → built the **layout and menus from scratch** in HTML·CSS·JS → learned a static page's basic structure first-hand
- **Time constraint (reflection):** Swapping templates was impossible past the commit limit, so it shipped incomplete → kept it as a **reflection** and set a retry goal → became the trigger for my "finish what I start" standard
- **Renewal (deploy):** I wanted to improve the incomplete result → **rebuilt the current portfolio site** on a new stack (Hugo Blox) and deployed it → completed as the site you're viewing now

## My Role — Solo Project

- Planning: planned the personal portfolio site
- Development: built it on a template (later renewed into the current site)
- Deploy: GitHub Pages

## Infra / Deploy

- Deployed the static site via **GitHub Pages**; the renewed version runs on a **GitHub Actions** automatic build & deploy pipeline
