**서비스 개요 —** 날씨 데이터를 감지해 빨래하기 좋은 시점을 알려주는 Raspberry Pi 5 기반 시스템입니다.

생활 속 문제(빨래 타이밍)를 **IoT 하드웨어 + 데이터 판단 로직**으로 풀어보는 개인 프로젝트로, 현재 진행 중입니다.

- **진행 환경:** 개인 프로젝트 · 2026.06 ~ (진행 중)
- **참여 인력:** 개인 프로젝트 (단독 진행)

## 프로젝트 상세 — 문제 → 해결 → 결과

- **빨래 타이밍 (생활 문제):** 언제 빨래를 널어야 잘 마를지 판단이 어려움 → 날씨 데이터를 **수집·판단하는 로직**을 구현 → 빨래하기 좋은 시점을 알려주는 것을 목표로 함
- **하드웨어 연동 (IoT):** 실제 환경 데이터를 받아와야 함 → **Raspberry Pi 5**에서 날씨 데이터를 수집하도록 구성 → 소프트웨어와 하드웨어를 잇는 경험을 쌓는 중

## 담당 역할 — 개인 프로젝트

- 기획: 날씨 데이터로 빨래하기 좋은 시점을 알려주는 시스템 구상
- 개발: Raspberry Pi 5에서 날씨 데이터 수집·판단 로직 구현 (Python)
